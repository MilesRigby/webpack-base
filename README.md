# webpack-base
A foundation for npm webpack projects, including minimal modules and functionality. Allows out of the box use of HTML, CSS, JavaScript, and ESLint.

Entry point: index.js
HTML base: template.html

### Setup
 - Run npm install
 - Run npm install --save-dev webpack-dev-server
 - Modify package.json fields - name; description; repository.url; bugs.url; homepage - to reflect new repository
 - Replace content in this README.md with new project info.
